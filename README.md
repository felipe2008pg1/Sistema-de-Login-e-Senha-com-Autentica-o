# SISTEMA DE LOGIN E SENHA COM AUTENTICAÇÃO | 🇧🇷 Português
### 📝 **Descrição**
- Este é um sistema de autenticação backend desenvolvido em Python, focado na gestão de usuários, validação de credenciais e persistência de dados. O projeto simula o fluxo real de uma aplicação, incluindo proteção contra ataques de força bruta (limite de tentativas) e manipulação de arquivos externos.

### 🛠️ **Funcionalidades**
- [Cadastro de Usuários]: Validação para evitar nomes duplicados no sistema.
- [Sistema de Login]: Verificação de credenciais com busca em base de dados.
- [Persistência de Dados (JSON)]: Armazenamento automático de usuários em arquivo .json, garantindo que os dados não sejam perdidos ao fechar o programa.
- [Segurança Básica]: Bloqueio temporário após 3 tentativas incorretas de login.
- [Utilidades]: Exibição de data/hora atual e relatório de banco de dados para testes.

_Interface via Terminal: Sistema limpo com funções de limpeza de tela dinâmicas (compatível com Windows e Linux)._

### 🚀 Tecnologias
- Python 3.x
- Biblioteca json (Manipulação de dados)
- Biblioteca os e sys (Integração com o sistema operacional)

# LOGIN AND PASSWORD SYSTEM WITH AUTHENTICATION | 🇺🇸 English
### 📝 Description
- This is a backend authentication system developed in Python, focused on user management, credential validation, and data persistence. The project simulates a real-world application workflow, including protection against brute-force attacks (attempt limit) and external file handling.

### 🛠️ Features
- [User Registration]: Validation to prevent duplicate names in the system.
- [Login System]: Credential verification with database search.
- [Data Persistence (JSON)]: Automatic user storage in a .json file, ensuring data is not lost when the program is closed.
- [Basic Security]: Temporary lockout after 3 incorrect login attempts.
- [Utilities]: Current date/time display and database reporting for testing.

_Terminal Interface: Clean system with dynamic screen clearing functions (Windows and Linux compatible)._

### 🚀 Technologies
- Python 3.x
- json library (Data manipulation)
- os and sys libraries (Operating system integration)
